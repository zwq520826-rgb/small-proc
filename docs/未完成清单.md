   - [ ] 生成收益日报/周报/月报
   - [ ] 收益数据导出（Excel/PDF）
   - [ ] 收益分析报告（订单完成率、平均收益等）

**数据库字段**：
- `wallets.total_income`：累计收入（已有）
- `wallets.total_expense`：累计支出（已有）
- `transactions`：交易记录表（已有，包含 `type: 'income'`）

**收益计算示例**：
```javascript
// 今日收益
const todayIncome = await db.collection('transactions')
  .where({
    user_id: uid,
    type: 'income',
    status: 'success',
    create_time: db.command.gte(todayStart)
  })
  .aggregate()
  .group({
    _id: null,
    total: $.sum('$amount')
  })
  .end()
```

**注意事项**：
- 收益统计需要考虑时区问题
- 收益计算需要考虑退款情况（订单取消、退款等）
- 收益数据需要与订单数据保持一致

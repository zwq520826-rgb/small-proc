# 查看云函数日志步骤

## 方法1：在 uniCloud Web 控制台查看

1. 打开浏览器，访问：https://unicloud.dcloud.net.cn/
2. 登录你的账号
3. 选择你的服务空间（应该是阿里云）
4. 左侧菜单：**云函数** → **日志**
5. 筛选条件：
   - 云函数名称：选择 `uni-id-co`
   - 时间范围：最近1小时
6. 点击 **查询**
7. 查看最新的错误日志（红色的）
8. 截图或复制错误信息

## 方法2：在 HBuilderX 中查看

1. 在 HBuilderX 顶部菜单：**视图** → **显示控制台**
2. 控制台底部切换到 **uniCloud** 标签
3. 查看是否有错误日志

## 需要关注的信息

查找包含以下关键词的日志：
- `loginByWeixin`
- `code2Session`
- `getAccessToken`
- `errcode`
- `40001`、`40003`、`40013` 等错误码

## 临时解决方案

如果查看日志比较麻烦，我们可以先让你的小程序能正常使用：

### 使用已登录的账号

如果你之前已经登录过，可以：
1. 跳过登录页面
2. 直接进入首页测试订阅消息功能

### 或者添加调试代码

我可以帮你在前端添加调试代码，直接显示错误信息。
